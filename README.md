# Expense Tracker

## Description
A Python program that helps users track their expenses, view all entries, and calculate the total amount spent.

## Features
- Add expenses
- View all expenses
- Calculate total spending
- Input validation with error handling

## Technologies Used
- Python
- Lists
- Loops
- Conditional Statements
- Exception Handling (try/except)

## How to Run
1. Install Python.
2. Download the project files.
3. Open terminal in the project folder.
4. Run:
python main.py
